# smart-homework-tracker
This project is a simple homework tracking application designed to help students stay organized by managing assignments, due dates, and completion status. 
Users can create, edit, and delete assignments, add due dates, and mark tasks as completed while viewing everything in one place. 
The project is currently in its early stages, with the repository and initial structure set up and development beginning on core features. 
It is being built using a standard tech stack that may include a frontend interface, backend logic, and a database for storing assignments. 
The application can be run locally by cloning the repository, opening it in an IDE such as Visual Studio, and running the project. 
Responsibilities are divided among team members to cover areas like backend setup, frontend design, core functionality, and testing. 
Future improvements may include adding notifications, a calendar view, improved user interface design, and user account support.
